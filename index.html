code park

Code Park opens fun coding experiences to students in the Houston area.

In our after-school classes, students focus on fundamental coding skills and work on real world projects. They get the hands-on experience they need to continue their own journeys as coders.

Our students will learn and build small projects with JavaScript, HTML, and CSS in Fall 2015.
